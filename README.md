# alx_travel_app
Initial Django project setup for a scalable travel listing platform. Includes MySQL database configuration, REST framework setup, CORS integration, and Swagger API documentation.


## Features Implemented in Milestone 1

- Django project setup with modular structure
- MySQL database integration using environment variables for credentials
- Django REST Framework setup for building APIs
- Cross-Origin Resource Sharing (CORS) configuration
- Swagger documentation via `drf-yasg` at `/swagger/`
- Prepared for future task queuing with Celery and RabbitMQ
